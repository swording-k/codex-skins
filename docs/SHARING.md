## Release Checklist

- `theme.json` and directory name use the same `preset-*` id.
- `background.jpg` is a pure wallpaper, not a Codex screenshot.
- Image has no readable text, logo, watermark, celebrity, or copyrighted character.
- Theme passes upstream payload validation.
- README includes install instructions and non-official disclaimer.
