## 主题要求

每个可分享主题建议包含：

- `theme.json`
- `background.jpg` 或 `background.png`
- `preview.png`
- `README.md`

上游真正读取的是 `theme.json` 里声明的背景文件。不要把 Codex 界面截图、按钮、文字、logo 或水印烘焙进 `background.*`。
